# Объектная среда для вероятностного анализа в системе MATLAB

Включает в себя следующие разделы:
- Объектное моделирование независимых случайных событий;
- раздел 2;
- раздел 3.

Пример:
Устройство состоит из N звеньев. 
Отказ i-го звена устройства - случайное событие Ai. 
Отказ устройства наступает в соответствии с формулой сложного события F. 
Вероятности событий A1…AN одинаковы и составляют R. 
Вычислить вероятность отказа устройства.
N=8; F='A1*A2+(A3+A4)\*(A5+A6)+A7\*A8'; R=0.31;

Решение:
N=8; F='A1\*A2+(A3+A4)\*(A5+A6)+A7\*A8'; R=0.31;
a=Randev( R\*ones(1, N) );
X=Set( a, F(F~='A') );
X=Value(X)
